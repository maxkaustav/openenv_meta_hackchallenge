# AI Healthcare Scheduling Agent

This is an OpenEnv-compatible RL environment for healthcare appointment scheduling.
It features an interactive Streamlit UI where you can test different agents (Groq LLM or Oracle).

## Setup
To run this locally:
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
